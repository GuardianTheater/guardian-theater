About

Guardian Theater is a mash up of Destiny activity history with publicly shared Xbox One Game DVR and Twitch gameplay clips.

Searching for a Guardian's Gamertag or PSN ID retrieves a list of recent activities that Guardian has participated in. Then for each activity, the site seeks out to display all gameplay clips recorded by any of the participating Guardians which overlap with the timespan in which the activity took place.

Guardian Theater was created by Chris Fried with invaluable assistance from Dustin Rue.

Just Xbox One? Why don't you support PS4?

UPDATE: As of November 13, 2016, with the addition of Twitch videos, we now support PS4 Guardians! It's a bit sparser than on the Xbox side, but it works!

In Xbox world, every clip a user records is uploaded to the cloud, stamped with the time it was recorded and the game that was being played. This makes it easy for us to pull in Destiny clips and compare them with Destiny activity timestamps pulled from Bungie.

In PlayStation world, clips are only available if a user chooses to upload them to an external website, such as YouTube. These clips are often tagged with the game they were played in, but they contain no information about when the clip was recorded. This renders support for PS4 clips on Guardian Theater next to impossible.

We'd support PS4 clips if we could. If you have any bright ideas on how we might, please get in touch!

This is cool, but what is it actually doing?

When you land on a Guardian's page, Guardian Theater asks Bungie's API for the 25 most recent activities the active character has participated in. You'll see a big spinning gear replaced by a list of greyed out activities with spinning gears.

For each activity visible on your screen, Guardian Theater will ask Bungie's API for that activity's Post Game Carnage Report. This contains the list of players. The spinning gear will dissappear and a list of players will appear below the activity.

Then for each player visible, Guardian Theater will ask the Xbox API for clips that player has recorded in Destiny. Guardian Theater will check if any of the clips received occurred during the activity. The player's name will dissappear, replaced by any matching clips that may have been found.

As you scroll down the page, more activities and players will come into view, triggering more players and clips to load. Reaching the bottom of the page will load in the next 25 most recent activities for the character.
